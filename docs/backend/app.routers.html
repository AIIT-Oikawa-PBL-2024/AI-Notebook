async app.routers.outputs_stream.request_content(files: list[str], db: AsyncSession = Depends(get_db)) → StreamingResponse[ソース]

ファイル名のリストを入力して、出力を生成するエンドポイントです。

パラメータ:

  • files (list[str]) -- ファイル名のリスト

  • db (AsyncSession) -- 非同期セッション

戻り値:

ストリーミングレスポンス

戻り値の型:

StreamingResponse

例外:

  • HTTPException 404 -- 指定されたファイルが見つからない場合

  • HTTPException 400 -- ファイル名の形式が無効な場合

  • HTTPException 500 -- コンテンツの生成中にエラーが発生した場合、 またはGoogle APIからエラーが返された場合、 またはコンテンツのストリーミング中にエラーが発生した場合
